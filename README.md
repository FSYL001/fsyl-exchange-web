# fsyl-exchange-web

> 一个个人写着玩的货币兑换网页工具  
> 主要是自己用，顺便放 GitHub 当个备份，防止哪天本地丢了。  
> 没啥技术含量，小白作品，请多包涵 😂

## ✨ 功能
- 在线获取汇率并进行货币转换
- 支持双向编辑（左/右输入框都可改）
- 一键反转货币顺序
- 自动计算（输入 0.2 秒后自动刷新结果）
- 支持多种常用货币
